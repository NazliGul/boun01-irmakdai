In this report electricity market prices (MCP & SMP) of July 2020 are analyzed using dplyr and ggplot2.

Background:

SMP: System marginal price
MCP: Market clearing price
MCPs are settled in a single session and they do not change.
SMP is always higher than MCP if system has Energy Deficit, and lower if there is Energy Surplus.
Positive imbalance price (TL/MWH) = 0.97xmin(PTF,SMF)
Negative imbalance price (TL/MWH) = 1.03xmax(PTF,SMF)

I noticed that comma was used as decimal seperator, I replaced all commas with points in necessary columns.

data$SMF <- gsub(",",".",data$SMF)
data$PTF <- gsub(",",".",data$PTF)
data$Pozitif.Dengesizlik.Fiyatı..TL.MWh. <- gsub(",",".",data$Pozitif.Dengesizlik.Fiyatı..TL.MWh.)
data$Negatif.Dengesizlik.Fiyatı..TL.MWh. <- gsub(",",".",data$Negatif.Dengesizlik.Fiyatı..TL.MWh.)

data$SMF <- as.numeric(data$SMF)
data$PTF <- as.numeric(data$PTF)
data$Pozitif.Dengesizlik.Fiyatı..TL.MWh. <- as.numeric(data$Pozitif.Dengesizlik.Fiyatı..TL.MWh.)
data$Negatif.Dengesizlik.Fiyatı..TL.MWh. <- as.numeric(data$Negatif.Dengesizlik.Fiyatı..TL.MWh.)

Since the column names are in Turkish, I translated them to English. Also I added seperate columns for date and time.

data_english <- data %>%
  rename(MCP=PTF, SMP=SMF, dt=Tarih, "positive_imbalance_price(TL/MWh)"=Pozitif.Dengesizlik.Fiyatı..TL.MWh.,"negative_imbalance_price(TL/MWh)"=Negatif.Dengesizlik.Fiyatı..TL.MWh.)%>%
  mutate(SMP_direction=case_when(SMF.Yön=="?Enerji Fazlası" ~ "Energy Surplus", SMF.Yön=="? Enerji Açığı"~ "Energy Deficit", SMF.Yön=="? Dengede" ~ "In Balance")) %>%
  mutate(dt=as.POSIXct(dt, format="%d.%m.%y %H:%M"), MCP_SMP_diff=MCP-SMP)%>%
  mutate(day=lubridate::day(dt),month=lubridate::month(dt),year=lubridate::year(dt),hour=lubridate::hour(dt),day_of_week=lubridate::wday(dt)) %>%
  select(dt,day,hour,day_of_week, MCP, SMP, "positive_imbalance_price(TL/MWh)","negative_imbalance_price(TL/MWh)",MCP_SMP_diff, SMP_direction)

I plotted hourly SMP and MCP to have a general idea of the data

ggplot(data_english, aes(x=dt))+geom_line(aes(y=SMP,color="darkred"))+geom_line(aes(y=MCP,color="navy"))+labs(x="Time", y="TL",title="SMP(System Marginal Price) and MCP(Market Clearing Price) in July 2020")+scale_color_discrete(name = "", labels = c("SMP", "MCP"))+theme_minimal()

Line chart of daily averages of SMP and MCP

average_by_day <- data_english %>%

Finally I wanted to see how SMP and MCP change throughout a day, so I took average for hours of day. It can be clearly seen that both SMP and MCP have a minimum value in the mornings (~6am).
